COMO INSTALAR
──────────────
  bash instalar.sh

La instalacion es completamente automatica (~3-5 minutos).
Compatible con Ubuntu, Debian, Fedora, Arch y derivados.

COMANDOS UTILES (despues de instalar)
───────────────────────────────────────
  cd ~/crypto_worker

  Ver estado       : ./status.sh
  Detener workers  : ./stop_workers.sh
  Iniciar workers  : ./start_workers.sh
  Actualizar       : ./update.sh
  Servicio systemd : systemctl --user status bittrader-workers
  Ver logs (W1)    : tail -f logs/worker_1.log

REQUISITOS MINIMOS
───────────────────
  Linux con kernel 4.x+ (Ubuntu 20.04+, Debian 11+, Fedora 35+)
  4 GB de RAM (8+ recomendado)
  Python 3.8+ (se instala automaticamente)
  Conexion a internet
